# Docs MCP v0.2.0 Standby-Native Refactor — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove the boot-time required `sources` validation and refactor `ServerState` so indexes are built lazily on first `tools/call` reference, enabling Standby containers to boot with no input.

**Architecture:** `ServerState.sources` becomes a `Map<string, SourceIndex>` (keyed by source name) instead of a frozen array. A `buildLocks` map holds in-flight build promises so concurrent first-requests for the same source share one network fetch. MCP tools gain an optional `sourceDef` argument that triggers a lazy index build when the named source isn't in the Map yet.

**Tech Stack:** TypeScript, Vitest, Apify SDK, Node.js `http`, no new dependencies.

## Spec Sharpening (resolve before writing tests)

These were open in the spec. Answers locked here:

1. **`sourceDef` JSON Schema for tool `inputSchema`:** same properties as `SourceConfig` — `name` (required, string), plus optional `template`, `url`, `sitemapUrl`, `contentSelector`. The `name` field is required because it becomes the Map key.

2. **`ensureIndexed` signature:**
   ```typescript
   async function ensureIndexed(state: ServerState, sourceDef: SourceDef): Promise<SourceIndex>
   ```
   - If `sourceDef.name` is already in `state.sources`, return it immediately.
   - If a build lock exists for `sourceDef.name`, await it (thundering-herd prevention).
   - Otherwise: create the lock promise, await it, store the resolved index in `state.sources`, **delete the lock** (on both resolve AND reject so retries are possible), then return the index.
   - On build failure: lock is deleted, the error propagates — caller (`dispatchTool`) catches it and returns `{ error: "Failed to build index for '<name>': <message>" }`.

3. **`source` remains required in `get_toc`, `search_docs`, `get_page`:** -32602 protocol error if `source` is missing AND `sourceDef` is also absent. `sourceDef` is a lazy-build hint; the lookup key is still `source` (or `sourceDef.name` when passed without `source`, which the dispatcher resolves as `sourceDef.name`). Simplest rule: if `sourceDef` is present and `source` is absent, treat `sourceDef.name` as the source name.

4. **Error message for unindexed source with no `sourceDef`:** `{ error: "Source not indexed: \"<name>\". Pass sourceDef to index it on demand." }`

5. **`maxPagesPerSource` per-request override:** out of scope for v0.2. `ensureIndexed` always uses `state.maxPagesPerSource`.

- [ ] **Step 4: Add `ensureIndexed` helper (above `handleMcpRequest`)**

```typescript
import { buildSourceIndex } from './indexer.js';

async function ensureIndexed(state: ServerState, sourceDef: SourceDef): Promise<SourceIndex> {
  const existing = state.sources.get(sourceDef.name);
  if (existing) return existing;

  const inflight = state.buildLocks.get(sourceDef.name);
  if (inflight) return inflight;

  const buildPromise = buildSourceIndex(sourceDef, state.fetchFn, state.maxPagesPerSource)
    .then((index) => {
      state.sources.set(index.name, index);
      state.buildLocks.delete(index.name);
      return index;
    })
    .catch((err: unknown) => {
      state.buildLocks.delete(sourceDef.name);
      throw err;
    });

  state.buildLocks.set(sourceDef.name, buildPromise);
  return buildPromise;
}
```

- [ ] **Step 6: Update `dispatchTool` to use Map + call `ensureIndexed`**

Full replacement of the `dispatchTool` function:

```typescript
async function dispatchTool(
  name: string,
  args: Record<string, unknown>,
  state: ServerState,
): Promise<unknown> {
  // Extract sourceDef if present (used for lazy indexing in get_toc, search_docs, get_page)
  const sourceDef = args['sourceDef'] as SourceDef | undefined;

  switch (name) {
    case 'list_sources':
      return {
        sources: Array.from(state.sources.values()).map((s) => ({
          name: s.name,
          url: s.url,
          pageCount: s.pages.length,
        })),
      };

    case 'get_toc': {
      const sourceName = args['source'] as string;
      if (sourceDef) {
        try {
          await ensureIndexed(state, sourceDef);
        } catch (e) {
          const msg = e instanceof Error ? e.message : String(e);
          return { error: `Failed to build index for "${sourceDef.name}": ${msg}` };
        }
      }
      const source = state.sources.get(sourceName);
      if (!source) {
        return {
          error: `Source not indexed: "${sourceName}". Pass sourceDef to index it on demand.`,
        };
      }
      return { source: source.name, pages: source.pages };
    }

    case 'search_docs': {
      const query = args['query'] as string;
      if (query.length > 500) return { error: 'query must be <= 500 characters' };
      const sourceFilter = typeof args['source'] === 'string' ? args['source'] : undefined;
      if (sourceDef) {
        try {
          await ensureIndexed(state, sourceDef);
        } catch (e) {
          const msg = e instanceof Error ? e.message : String(e);
          return { error: `Failed to build index for "${sourceDef.name}": ${msg}` };
        }
      }
      const maxResults = Math.min(
        typeof args['maxResults'] === 'number' ? Math.max(1, Math.floor(args['maxResults'])) : 10,
        30,
      );
      const results: SearchResult[] = searchDocs(
        query,
        Array.from(state.sources.values()),
        state.cache,
        { maxResults, sourceFilter },
      );
      return { query, source: sourceFilter ?? null, results };
    }

    case 'get_page': {
      const url = args['url'] as string;

      if (sourceDef) {
        try {
          await ensureIndexed(state, sourceDef);
        } catch (e) {
          const msg = e instanceof Error ? e.message : String(e);
          return { error: `Failed to build index for "${sourceDef.name}": ${msg}` };
        }
      }

      const cached = state.cache.get(url);
      if (cached) {
        return {
          url: cached.url,
          title: cached.title,
          source: cached.source,
          content: cached.content,
          cachedAt: cached.cachedAt,
        };
      }

      const sourceIndex = Array.from(state.sources.values()).find((s) =>
        s.pages.some((p) => p.url === url),
      );
      if (!sourceIndex) return { error: 'URL not found in any configured source index' };

      const page = sourceIndex.pages.find((p) => p.url === url)!;
      const fetchResult = await state.fetchFn(url);
      if (!fetchResult.ok) {
        return { error: `Failed to fetch page: HTTP ${fetchResult.status}`, url };
      }

      const content = extractContent(fetchResult.html, sourceIndex.contentSelector, state.markdownOutput);
      const cachedPage: CachedPage = {
        url,
        title: page.title,
        source: sourceIndex.name,
        content,
        cachedAt: new Date().toISOString(),
      };
      state.cache.set(url, cachedPage);

      return {
        url,
        title: cachedPage.title,
        source: cachedPage.source,
        content: cachedPage.content,
        cachedAt: null,
      };
    }

    default:
      return { error: `Unknown tool: ${name}` };
  }
}
```

```typescript
import { Actor, log } from 'apify';
import http from 'node:http';
import type { ActorInput, CachedPage, FetchFn } from './types.js';
import { buildSourceIndex } from './indexer.js';
import { LruCache } from './cache.js';
import { handleMcpRequest, type ServerState } from './mcp.js';

async function fetchUrl(url: string): Promise<{ html: string; ok: boolean; status: number }> {
  const response = await fetch(url, {
    headers: { 'User-Agent': 'docs-mcp-server-starter/0.1 (Apify Actor)' },
  });
  const html = await response.text();
  return { html, ok: response.ok, status: response.status };
}

function validateInput(raw: unknown): ActorInput {
  const input = (raw ?? {}) as ActorInput;
  const sources = input.sources ?? [];
  if (sources.length > 10) throw new Error('sources: maximum 10 allowed.');
  for (const src of sources) {
    if (!src.name) throw new Error('Each source must have a name.');
    if (!src.template && (!src.url || !src.contentSelector)) {
      throw new Error(`Source "${src.name}": requires template OR (url + contentSelector).`);
    }
  }
  const maxPagesPerSource = input.maxPagesPerSource ?? 200;
  const cacheMaxPages = input.cacheMaxPages ?? 50;
  if (maxPagesPerSource < 1 || maxPagesPerSource > 500) throw new Error('maxPagesPerSource must be 1–500.');
  if (cacheMaxPages < 1 || cacheMaxPages > 200) throw new Error('cacheMaxPages must be 1–200.');
  return {
    sources,
    maxPagesPerSource,
    cacheMaxPages,
    markdownOutput: input.markdownOutput ?? true,
  };
}

await Actor.init();

try {
  const raw = await Actor.getInput();
  const input = validateInput(raw);

  const fetchFn: FetchFn = fetchUrl;
  const sourceMap = new Map<string, import('./types.js').SourceIndex>();

  if (input.sources.length > 0) {
    log.info('Pre-building source indexes…', { sources: input.sources.map((s) => s.name) });
    const built = await Promise.all(
      input.sources.map((s) => buildSourceIndex(s, fetchFn, input.maxPagesPerSource!)),
    );
    for (const idx of built) sourceMap.set(idx.name, idx);
    log.info('Indexes ready.', {
      totalPages: built.reduce((n, s) => n + s.pages.length, 0),
      sourceCount: built.length,
    });
  } else {
    log.info('No boot-time sources configured. Standby mode: indexes built on first tool call.');
  }

  const cache = new LruCache<string, CachedPage>(input.cacheMaxPages!);
  const state: ServerState = {
    sources: sourceMap,
    cache,
    fetchFn,
    markdownOutput: input.markdownOutput!,
    maxPagesPerSource: input.maxPagesPerSource!,
    buildLocks: new Map(),
  };

  const port = parseInt(process.env['ACTOR_STANDBY_PORT'] ?? '4321', 10);

  const server = http.createServer(async (req, res) => {
    if (req.method !== 'POST') {
      res.writeHead(405, { 'Content-Type': 'text/plain' }).end('Method Not Allowed');
      return;
    }
    let body = '';
    req.on('data', (chunk: Buffer) => { body += chunk.toString(); });
    req.on('end', async () => {
      try {
        const request = JSON.parse(body) as Parameters<typeof handleMcpRequest>[0];
        const response = await handleMcpRequest(request, state);
        res.writeHead(200, { 'Content-Type': 'application/json' });
        res.end(JSON.stringify(response));
      } catch {
        res.writeHead(400, { 'Content-Type': 'text/plain' }).end('Bad Request');
      }
    });
  });

  await new Promise<void>((resolve, reject) => {
    server.listen(port, () => log.info('MCP server listening.', { port }));
    server.on('error', reject);
    process.on('SIGTERM', () => server.close(() => resolve()));
  });
} catch (e) {
  const message = e instanceof Error ? e.message : String(e);
  log.error('Actor failed.', { message });
  await Actor.fail(message);
}

await Actor.exit();
```

#!/usr/bin/env bash
# Production smoke test for the Docs MCP Server Starter running on Apify Standby.
# Points at the live actor URL; verifies 6 tool calls succeed with sourceDef.
#
# Usage: STANDBY_URL=https://<actor-id>.apify.actor APIFY_TOKEN=<token> ./scripts/smoke-test-prod.sh

set -euo pipefail

STANDBY_URL="${STANDBY_URL:?Set STANDBY_URL to the actor standby endpoint}"
APIFY_TOKEN="${APIFY_TOKEN:?Set APIFY_TOKEN}"
AUTH_URL="${STANDBY_URL}?token=${APIFY_TOKEN}"

pretty() {
  if command -v jq >/dev/null 2>&1; then jq .; else cat; fi
}

call_mcp() {
  local label="$1" payload="$2"
  echo
  echo "── $label ──"
  curl -sS -X POST "$AUTH_URL" \
    -H 'Content-Type: application/json' \
    -d "$payload" | pretty
}

call_mcp "initialize" '{
  "jsonrpc": "2.0", "id": 1, "method": "initialize",
  "params": { "protocolVersion": "2024-11-05", "capabilities": {}, "clientInfo": { "name": "smoke-test-prod", "version": "0.2.0" } }
}'

call_mcp "tools/list" '{
  "jsonrpc": "2.0", "id": 2, "method": "tools/list"
}'

call_mcp "list_sources" '{
  "jsonrpc": "2.0", "id": 3, "method": "tools/call",
  "params": { "name": "list_sources", "arguments": {} }
}'

call_mcp "get_toc (Next.js Docs via sourceDef)" '{
  "jsonrpc": "2.0", "id": 4, "method": "tools/call",
  "params": {
    "name": "get_toc",
    "arguments": {
      "source": "Next.js Docs",
      "sourceDef": { "name": "Next.js Docs", "template": "nextjs" }
    }
  }

call_mcp "search_docs (routing)" '{

      "source": "Next.js Docs",
      "sourceDef": { "name": "Next.js Docs", "template": "nextjs" },
      "maxResults": 3
    }
  }
}'

echo
